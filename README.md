# Frontend-Catalog

Frontend Angular para OmniBacklog, un catalogador personal de videojuegos.

## Tech Stack

- Angular 21
- TypeScript
- Signals (estado reactivo)
- Standalone Components

## Requisitos

- Node 22 (via nvm)

## Levantar

```bash
npm install
npx ng serve
```

Corre en `http://localhost:4200`. Requiere el backend corriendo en `http://localhost:8080`.
